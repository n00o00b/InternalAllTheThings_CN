# aka.ms 快捷方式 (Shortcuts)

aka.ms 是微软使用的 URL 缩短服务。它通常用于创建简短且易于分享的链接，将用户重定向到更长或更复杂的 URL，这些 URL 通常与微软的服务、产品或资源相关。
